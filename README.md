<h1 align="center">KromaNet - KromaRec's website</h1>

<p align="center">
  Official web platform for the KromaRec ecosystem
</p>

<p align="center">
  A modern web hub for documentation, downloads, updates, and community access for all KromaRec projects.
</p>

<hr>

<h3 align="center">About</h3>

<p align="center">
  KromaNet is the central web platform of the KromaRec ecosystem.  
  It provides information, documentation, downloads, and links to all related projects including the server, client, and Discord bot.
</p>

<p align="center">
  The goal is to create a clean, fast, and unified experience for users and developers.
</p>

<hr>

<h3 align="center">Features</h3>

<p align="center">
  Project documentation<br>
  Download center for builds<br>
  News and updates<br>
  Ecosystem navigation hub<br>
  Future API and developer portal
</p>
